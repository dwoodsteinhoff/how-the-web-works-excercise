How the Web Works

What is HTTP?

HTTP = Hyper Text Transfer Protocol

It is how to communicate between servers and browsers. How they get and send data to a server.


What is a URL?

URL = Uniform Resource Locator

It is the way to access an internet resource. It has a protocol, hostname, port (unviewable in the URL), a resource, and then a query.


What is DNS

DNS = Domain Name System

they take hostname(s) or URLs and translate it/them into a corresponding ip address


What is a query string?

It is extra information and is typically used in search bars to indicate the extra information. It is typically done like ?x=1&x=2 which are key-value pairs.


list two HTTP Verbs and their use cases

  • GET - Get data from a server
  • POST - Send data to a server (often times they change the data on the server.)

what is an HTTP request?

A request from a browser to a server, using the HTTP protocol.


what is an HTTP response?

A response from a server to the browser, using the HTTP protocol.

    • What happens when you type a URL in a browser?

    1. The Browser turns the url or hostname into an IP address through the DNS
    2. The Browser then makes a request to that IP address
    3. The server responds to the request hopefully with a status code of 200 and not a 404(error:not found code)
    4. The browser makes the web page using the HTML and other resources located in the response
    5. If there are other resources, then the browser sends requests for those resources awaiting a response from the server
